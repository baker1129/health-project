# 日次入力テンプレート

以下をコピーして Claude との会話に貼るだけで、各ファイルに自動反映されます。

---

## テンプレート

```
【日次記録】YYYY-MM-DD
体重: XX.Xkg / 体脂肪: XX.X%
朝血圧: 収縮期/拡張期/脈拍, 収縮期/拡張期/脈拍 / cpap:on
夜血圧: 収縮期/拡張期/脈拍, 収縮期/拡張期/脈拍
朝食: 内容
昼食: 内容
夕食: 内容
気づき: 自由記述
運動: 内容
```

---

## 記入ルール

| 項目 | ルール |
|---|---|
| 日付 | `YYYY-MM-DD` 形式 |
| 体脂肪 | 測定しない日は `/ 体脂肪: XX.X%` ごと省略可 |
| 血圧 | `収縮期/拡張期/脈拍` の順。2回計測は `,` で区切る。1回のみでも可 |
| cpap | `cpap:on` または `cpap:off`。朝血圧の行末に付ける。省略した場合は未記録扱い |
| 朝食/昼食/夕食 | 食べていない場合は `なし` と書くか行ごと省略 |
| 気づき | 省略可 |
| 運動 | していない場合は `なし` |
| 朝血圧/夜血圧 | 測定しなかった場合は行ごと省略 |

---

## 記入例

```
【日次記録】2026-05-03
体重: 91.9kg / 体脂肪: 32.4%
朝血圧: 131/81/77, 133/87/74 / cpap:on
夜血圧: 122/81/114, 118/80/111
朝食: なし
昼食: サラダ、鶏汁、のりチキン、角煮たまご
夕食: 白飯小盛り、納豆
気づき: 食後散歩15分
運動: ウォーキング 30分
```
